# Deploy Java Spring Boot Backend on Amazon EC2

#### Overview

In this workshop, you will learn how to deploy a production-ready **Java Spring Boot RESTful API** for the **Pet Resort & Care System** directly onto **Amazon EC2** instances. The application connects to **Amazon RDS (MySQL)** for data persistence, uses **Amazon ElastiCache (Redis)** for caching, and traffic is routed through an **Application Load Balancer (ALB)**.
