### Week 3 Objectives:
- Learn the mechanics of cloud object storage and static asset management.
- Initialize a test storage environment for project assets (pet images, service invoices) for the Pet Resort application.
- Establish initial account monitoring and cost allocation alarms.

### Week 3 Achievements:

* Successfully activated cost-control monitoring via AWS Budgets to mitigate unexpected cloud expenditures.
* Initialized an isolated repository `petshop-media-storage` on Amazon S3, familiarizing with workflows for uploading and managing static assets.
* Mastered baseline permission controls and implemented safe IAM security configurations aligned with AWS best practices.
