This worklog documents a 13-week internship journey at **AWS First Cloud Journey** (April - July 2026). Throughout this period, our team built and deployed the **Pet Resort & Care System** — a web application combining pet e-commerce and spa booking — onto the AWS Cloud platform.

**Week 1:** [Getting familiar with the AWS ecosystem and practicing creating a basic static website using Hugo](1.1-week1/)

**Week 2:** [Learning about security and access management using IAM and IAM Roles](1.2-week2/)

**Week 3:** [Initializing virtual network infrastructure with VPC and creating Subnets](1.3-week3/)

**Week 4:** [Setting up secure internal network routing using NAT Gateway and VPC Endpoints](1.4-week4/)

**Week 5:** [Deploying Amazon S3 as storage for Frontend source code and Media files](1.5-week5/)

**Week 6:** [Accelerating content delivery with CloudFront (CDN) and configuring AWS WAF firewall](1.6-week6/)

**Week 7:** [Configuring Amazon EC2 virtual servers to prepare the Backend runtime environment](1.7-week7/)

**Week 8:** [Distributing network traffic with Application Load Balancer (ALB) and auto-scaling with EC2 Auto Scaling](1.8-week8/)

**Week 9:** [Setting up Amazon RDS database (Primary/Standby) and optimizing query performance with ElastiCache](1.9-week9/)

**Week 10:** [Managing security (KMS, Secrets Manager), system monitoring (CloudWatch), and setting up alerts (SNS)](1.10-week10/)

**Week 11:** [Systematizing knowledge, drawing the overall architecture, and planning data flow connections](1.11-week11/)

**Week 12:** [Initializing the network environment and performing actual deployment to AWS across Frontend, Backend, and Data tiers](1.12-week12/)

**Week 13:** [Performing Load Testing, bug fixing, configuration optimization, and final product launch](1.13-week13/)
