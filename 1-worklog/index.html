Worklog

This worklog documents a 13-week internship journey at AWS First Cloud Journey (April - July 2026). Throughout this period, our team built and deployed the Pet Resort & Care System — a web application combining pet e-commerce and spa booking — onto the AWS Cloud platform.

Week 1: Getting familiar with the AWS ecosystem and practicing creating a basic static website using Hugo

Week 2: Learning about security and access management using IAM and IAM Roles

Week 3: Initializing virtual network infrastructure with VPC and creating Subnets

Week 4: Setting up secure internal network routing using NAT Gateway and VPC Endpoints

Week 5: Deploying Amazon S3 as storage for Frontend source code and Media files

Week 6: Accelerating content delivery with CloudFront (CDN) and configuring AWS WAF firewall

Week 7: Configuring Amazon EC2 virtual servers to prepare the Backend runtime environment

Week 8: Distributing network traffic with Application Load Balancer (ALB) and auto-scaling with EC2 Auto Scaling

Week 9: Setting up Amazon RDS database (Primary/Standby) and optimizing query performance with ElastiCache

Week 10: Managing security (KMS, Secrets Manager), system monitoring (CloudWatch), and setting up alerts (SNS)

Week 11: Systematizing knowledge, drawing the overall architecture, and planning data flow connections

Week 12: Initializing the network environment and performing actual deployment to AWS across Frontend, Backend, and Data tiers

Week 13: Performing Load Testing, bug fixing, configuration optimization, and final product launch
